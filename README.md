Paradise Nursery is a simple e-commerce web application for purchasing houseplants.

## Features
- Browse plants by category
- View plant name, image, and price
- Add plants to shopping cart
- Increase or decrease item quantity
- Remove items from cart
- Dynamic cart count and total price
- Navigation bar with Home, Plants, and Cart pages

## Tech Stack
- React
- Redux Toolkit
- React Router
- Vite

This project was built as part of a final assignment to demonstrate React and Redux fundamentals.
